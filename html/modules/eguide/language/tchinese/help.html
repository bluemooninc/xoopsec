報名申請表格項目編輯

 受理報名 時所要出現的表單輸入方式 可以任意指定新增需要的項目與設定。
每一行為一個項目

可選擇種類有text、checkbox、radio、textarea、select、hidden、const
引數：有「+」和「=」

  • 加上"*" 為必填項目。
  • 加上"#" 可於表格後方加上描述或是備註。
  • 利用"," 為區隔指令的記號。如需要再引數里使用這個符號必須以HTML的特殊記號。
  • "+" 為 (checkbox, radio, select) 此三項的預選項目(checked)。

範例：

新增內容 表單形態
姓名* 姓名*
姓名*,size=5 姓名*
姓名*,size=5,#備註 姓名* 備註
選項,radio,項目1+,項目2,項目3 選項 項目1   項目2   項目3  
選項,checkbox,項目1+,項目2,項目3 選項 項目1   項目2   項目3  
選項,select,項目1,項目2,項目3 選項
項目名,const,値 項目名 値 (The value display and handle to input value)
項目名,hidden,値 (not display in form - present only input values)
#說明文或備註 說明文或備註
